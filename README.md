# IRIS

IRIS is a voice-first AI assistant and operating-system layer that listens, reasons, remembers context, and takes action on the user's machine.

## Authors

- Aradhya
- Aryan
- Maneesh

## Platform Support

IRIS is built for:

- Windows 10 / 11
- macOS

Linux remains a future or partial-support target depending on the module.

## Core Capabilities

- Voice input with wake-word driven interaction
- Cloud ASR and TTS with agent-based task execution
- Memory-aware planning and action routing
- Browser, coding, file, and desktop control workflows
- Transparent Tauri overlay UI

## Self-Improving Behavior

IRIS is designed to be self-improving. With the coding-agent path enabled, it can inspect its own repository, modify its own files, run tests, and iterate on improvements after user interactions when the task requires it and the user permits the change.
