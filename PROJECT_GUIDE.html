📖 Project Overview

What is IRIS?

IRIS is an open-source voice-controlled AI assistant designed for Windows that bridges the gap between natural language commands and system actions. It combines:

  • Voice Input — Speak naturally; IRIS listens and understands
  • AI Reasoning — DeepSeek/Claude-powered decision making
  • System Actions — Execute files, apps, browser automation, emails, tasks
  • Memory — Learns from past interactions and maintains context
  • Real-time UI — Transparent overlay with state feedback

Use Cases

  • Productivity — Automate repetitive tasks, manage todos, schedule reminders
  • Information — Weather, news, stock prices, web search
  • Desktop Control — Launch apps, manage files, execute scripts
  • Browser Automation — Fill forms, navigate sites, scrape data
  • Communication — Send emails, schedule meetings, manage messages

Current Status

  • ✅ Phase 1 Complete (core audio, LLM, agents, basic actions)
  • ✅ Phase 2 Complete (browser, coding agent, credential encryption, CLI)
  • ✅ Phase 3-4 Complete (email, todo, weather, timers)
  • 🟡 Pre-Launch In Progress (performance optimization, error handling, testing)

🏗️ Core Architecture

High-Level Flow

User (Voice)
    ↓
Audio Module (capture, wake word, interrupt)
    ↓
ASR Engine (Groq Whisper) → Text
    ↓
Core Event Loop (state manager, router)
    ↓
LLM Router (DeepSeek/Qwen) → Action
    ↓
Agent System (Planner, Executor, Coder)
    ↓
Action Router (validate, check safety, execute)
    ↓
Handler Dispatch (25+ action handlers)
    ↓
Memory System (store outcome)
    ↓
Voice/UI Output (TTS, overlay)
    ↓
User (Sees/Hears)

Key Design Principles

  1. Async-First — All I/O is non-blocking (audio, API, file)
  2. Safety by Default — Dangerous actions require approval
  3. Graceful Degradation — Works offline with fallbacks
  4. Memory-Aware — Learns from past interactions
  5. Modular — Easy to add new actions/agents
  6. Privacy-First — No data sent to cloud (except APIs you authorize)
  7. Cross-Platform — Paths, processes, APIs abstracted
  8. Testable — Unit + integration tests throughout

🚀 Installation

1. Prerequisites

python --version    # Should be 3.11+
git --version
rustup --version
cargo --version

2. Clone Repository

cd C:\Users\YourName\Desktop
git clone https://github.com/Aradhya648/Iris.git
cd Iris

3. Create Virtual Environment (Optional)

python -m venv venv
venv\Scripts\Activate.ps1

4. Install Dependencies

pip install -r requirements.txt
python -m playwright install chromium

5. Install Tauri CLI (for UI)

cargo install tauri-cli
# Takes ~8 minutes

6. Configure API Keys

Edit configs/settings.yaml and add:

llm:
  deepseek_api_key: "your_key_here"

voice:
  elevenlabs_api_key: "your_key_here"

audio:
  groq_api_key: "your_key_here"

weather:
  openweather_api_key: "your_key_here"

🎤 Usage Guide

Starting IRIS

Terminal 1 — Python Backend:

cd Iris
python main.py

Terminal 2 — UI Overlay (Optional):

cd Iris\ui\tauri-app
cargo tauri dev

Using IRIS

  1. Say "Iris" or "Jarvis" (wake word)
  2. Wait for blue border (listening state)
  3. Speak your command naturally
  4. IRIS processes and responds
  5. Green border = processing | White = done

Example Commands

"What time is it?"
"Open Notepad"
"Send email to bob@example.com, subject: Hello, body: Hi Bob"
"Check my email"
"Add task: finish project"
"What's the weather in Paris?"
"Set timer for 10 minutes"
"Remind me to check email in 30 minutes"
"Take a screenshot"
"Create file: notes.txt with content: Meeting tomorrow at 2pm"

State Feedback

Border State Meaning
White (static) IDLE Waiting for wake word
Blue (pulse) INTERACTIVE Listening to you
Green (sweep) ACTING Processing & executing
Fading white STOPPING Shutting down

👨‍💻 Development Guide

Adding a New Action (5 Steps)

Step 1: Create action module

# actions/my_action.py
async def my_action(param1: str) -> dict:
    """Do something."""
    try:
        result = "Success!"
        logger.info(result)
        return {"status": "ok", "result": result}
    except Exception as e:
        logger.error(str(e))
        return {"status": "error", "result": str(e)}

Step 2: Register in router

# actions/action_router.py
from actions import my_action

ACTION_HANDLERS = {
    ...
    "my_action": my_action.my_action,
}

Step 3: Classify safety

# actions/safety.py
ACTION_SAFETY_MAP = {
    ...
    "my_action": SafetyLevel.SAFE,  # or WARN/DANGEROUS
}

Step 4: Write tests

# tests/test_my_action.py
def test_my_action_registered():
    assert "my_action" in ACTION_HANDLERS

def test_my_action_safety():
    assert classify("my_action") == SafetyLevel.SAFE

Step 5: Document

Create docs/MY_ACTION_SETUP.md with usage guide.

Running Tests

python -m pytest tests/ -v
python -m pytest tests/test_email.py -v
python -m pytest tests/ --cov=actions

🔧 Troubleshooting

Common Issues

"Wake word not detecting"

Solutions:
  • Check microphone: Settings → Privacy → Microphone
  • Speak clearly and loudly
  • Restart IRIS

"DeepSeek API key not found"

Solution:
  • Add to configs/settings.yaml:
llm:
  deepseek_api_key: "sk_xxxxx"

"TTS not producing audio"

Solutions:
  • Check speaker volume
  • Verify ElevenLabs key in voice.elevenlabs_api_key
  • Check audio device settings

"Module not found"

Solution: 
pip install -r requirements.txt

Logs Location

Check logs/iris.log for full activity log.

❓ FAQs

Licensing & Cost

Is IRIS free?

Yes, IRIS is MIT licensed (open source, free to use/modify/distribute).

Do I need to pay for APIs?

Optional APIs (DeepSeek, ElevenLabs, Groq, OpenWeatherMap) have free tiers. IRIS works offline with local Qwen2.5.

Privacy & Security

Does IRIS send my data to the cloud?

Only to APIs you explicitly authorize (DeepSeek, ElevenLabs, etc.). All local processing stays on your PC.

Are my files/passwords safe?

Credentials are encrypted with the cryptography library. File access is local only. Browser passwords are encrypted in ~/.iris/credentials.enc.

Compatibility

Can IRIS work without internet?

Partially. Local Qwen2.5 + local Whisper (optional) work offline. Cloud APIs require internet.

Does IRIS work on Mac?

Not yet (Phase 6). Python code is cross-platform; Tauri UI needs setup.

Does IRIS work on Linux?

Partial (command-line only). Full Tauri support planned for Phase 6.
