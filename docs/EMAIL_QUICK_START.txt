                    IRIS EMAIL ACTIONS — QUICK START
================================================================================

NEW FEATURES ADDED:
  - Send emails via voice
  - Check unread email count

SETUP (5 minutes):

1. Edit: configs/settings.yaml

   Add this section:

   email:
     smtp_server: "smtp.gmail.com"
     smtp_port: 587
     sender_email: "your_email@gmail.com"
     sender_password: "your_password"     ← ADD YOUR GMAIL PASSWORD HERE
     imap_server: "imap.gmail.com"
     receiver_email: "your_email@gmail.com"
     receiver_password: "your_password"

2. If using Gmail:
   - Enable 2-step verification (security settings)
   - Create app-specific password (account.google.com)
   - Use that password in settings.yaml

3. Done! Restart IRIS.

TROUBLESHOOTING:
================================================================================

"Email not configured"
  → Add sender_email & sender_password to settings.yaml

"Authentication failed"
  → Gmail: Use app-specific password, not account password
  → Outlook: Enable 2-step verification first
  → Check email address matches

"Connection timeout"
  → Check SMTP/IMAP server is correct for your provider
  → Verify port 587 is not blocked by firewall

See docs/EMAIL_SETUP.md for detailed instructions.
